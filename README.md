# Binary Game Remake
A remake of a game that we made back in high school.

# Controls
- Numpad
- Mouse

# Installation (Windows)
- Go to Releases section
- Click on BinaryGameRemake1.0.zip
- Save it wherever you want
- Extract all the content
- Run BinaryGameRemake.exe
